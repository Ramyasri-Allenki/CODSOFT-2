# Project Documentation
## Tic-Tac-Toe AI Using Minimax With Alpha-Beta Pruning

---

## 1. Introduction
This project implements a human vs AI Tic-Tac-Toe game.  
The AI uses the Minimax algorithm with Alpha-Beta Pruning, ensuring it never loses.  
The purpose of the project is to understand game trees, search algorithms, and AI-based decision making.

---

## 2. Objective
- To build an unbeatable AI for Tic-Tac-Toe  
- To implement Minimax and Alpha-Beta pruning  
- To demonstrate adversarial search  
- To create a simple, interactive console-based game

---

## 3. Tools and Technologies Used
- **Language:** Python 3.8+  
- **Concepts:** Minimax, Alpha-Beta Pruning, Game Trees  
- **Paradigm:** Object-Oriented Programming  
- **Platform:** Console/Terminal  

---

## 4. System Overview

### 4.1 Game Board  
The board handles:
- Player moves  
- Tracking empty positions  
- Checking winners  
- Displaying the grid  

### 4.2 AI Agent  
Responsibilities:
- Evaluating possible outcomes  
- Simulating future moves  
- Choosing the best optimal move  
- Using Minimax + Alpha-Beta Pruning  

---

## 5. Minimax Algorithm
The Minimax algorithm simulates all possible game states:  
- The AI maximizes score  
- The human minimizes score  

This ensures the AI always chooses the strongest move.

---

## 6. Alpha-Beta Pruning
This optimization reduces unnecessary Minimax calculations, making the AI faster without changing results.

---

## 7. Scoring System
- **AI wins:** `+10 - depth`  
- **Human wins:** `-10 + depth`  
- **Draw:** `0`  

Depth ensures:
- Faster wins are preferred  
- Slower losses are preferred
